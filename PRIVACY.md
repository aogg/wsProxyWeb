# Privacy Policy for WebSocket Proxy Extension

**Last Updated: March 7, 2026**

## Overview

This privacy policy describes how the WebSocket Proxy Extension ("the Extension") handles user data.

## Data Collection

The Extension does NOT collect, store, or transmit any personal information to external servers.

## Local Data Storage

The Extension stores the following data locally on your device:
- WebSocket proxy server configuration (host, port)
- Connection preferences and settings

All data is stored locally using Chrome's storage API and never leaves your device.

## Data Usage

- Configuration data is used solely to establish WebSocket proxy connections
- No browsing history, personal information, or user activity is collected
- No analytics or tracking mechanisms are implemented

## Data Sharing

The Extension does NOT share any data with third parties. All operations are performed locally on your device.

## Permissions

The Extension requires the following permissions:
- `storage`: To save your proxy configuration locally
- `webRequest`: To intercept and proxy web requests
- Network access: To establish WebSocket connections to your specified proxy server

## Changes to This Policy

Any updates to this privacy policy will be reflected in the Extension's repository and updated version.

WebSocket代理 - 安全高效的网页请求代理工具

【简介】
WebSocket代理是一款通过WebSocket协议代理网页HTTP/HTTPS请求的浏览器扩展。支持加密传输、数据压缩和灵活的请求过滤规则，可配合frp等内网穿透工具实现安全访问内网服务。

【核心功能】
✓ 自动拦截并代理网页请求 - 支持GET、POST、PUT、DELETE等HTTP方法
✓ 军事级加密传输 - 支持AES-256-GCM和ChaCha20-Poly1305加密算法
✓ 智能数据压缩 - 支持Gzip和Snappy压缩，减少传输流量
✓ 灵活过滤规则 - 域名白名单/黑名单、URL模式匹配
✓ 完善安全控制 - 连接数限制、频率限制、请求大小限制
✓ 高性能设计 - HTTP连接池复用，支持高并发请求处理

【适用场景】
• 内网穿透访问 - 配合frp搭建HTTPS代理，安全访问内网服务器
• 开发调试 - 代理本地开发环境的API请求
• 网络测试 - 测试不同网络环境下的应用表现
• 请求分析 - 监控和分析网页请求流量

【使用方法】
1. 部署WebSocket代理服务器（支持Docker一键部署）
2. 安装本扩展并配置服务器地址
3. 根据需要配置加密、压缩和过滤规则
4. 启用请求拦截，扩展将自动代理符合规则的请求

【技术特性】
• 基于Manifest V3标准开发
• 支持Chrome 88+和Edge 88+浏览器
• 开源项目，代码透明可审计
• 完整的配置界面，无需修改代码

【安全提示】
⚠ 建议生产环境启用加密功能
⚠ 定期更换加密密钥
⚠ 合理配置IP和域名白名单
⚠ 不要在不可信网络中使用
